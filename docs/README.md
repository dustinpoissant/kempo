# Kempo Documentation

Kempo is a fullstack CMS framework. Install it into a project, scaffold the starter files, and you have a working website with user authentication, an admin panel, content management, and an extension system.

## Contents

- [Getting Started](getting-started.md) — installation, database setup, first run
- [Project Structure](project-structure.md) — directory layout and configuration files
- [Content Management](content-management.md) — pages, templates, fragments, and global content
- [Users, Auth & Permissions](users-auth-permissions.md) — accounts, login, groups, and access control
- [Settings](settings.md) — runtime key-value configuration
- [Extensions](extensions/README.md) — installing and managing extensions
- [Creating Extensions](extensions/creating-extensions.md) — guide for building your own extension
