Description

The directory viewer allows the user to select a directory from their local filesystem, and the component allows them to open/close subdirectories and select files.

Basic Usage

<k-directory-viewer></k-directory-viewer>

Constructor

Extends LazyComponent
new DirectoryViewer()

Requirements

  • class LazyComponent
  • class Component
  • class Icon
  • util element.js
  • util watchDirectoryHandle.js

Methods

async selectDirectory(): undefined

Opens the directory selection dialog. When a dialog is selected it will render the directory's contents.

open(): undefined

Opens the directory, this should only be called on auto-generated sub-directories.

close(): undefined

Closes the directory, this should only be called on auto-generated sub-directories.

toggleOpen(): undefined

Opens/Closes the directory, this should only be called on auto-generated sub-directories.

clearDirectoryHandle(): undefined

Clears the handle that represents the user selected directory, allowing the user to choose a new directory.
