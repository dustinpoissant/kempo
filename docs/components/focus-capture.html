Description

The FocusCapture class extends the LazyComponent class and provides functionality to capture focus within the component.

Basic Usage

Use the FocusCapture component to capture focus within the component. When the focus moves out of the component, it will automatically move back to the first focusable element inside the component.

<k-focus-capture>
  <button>Button 1</button>
  <button>Button 2</button>
</k-focus-capture>
