Code Editor

Use the <k-code-editor> to create a code text editor. Put your initial code within the the tag, for HTML you should escape the open tags (< should be &lt;) so the code is not treated as real HTML.

This is a very simple code editor, it is not intended to be full featured, just the basics, it's like a textarea with line numbers and highlighting. If you want a full featured editor please use Ace Editor, Code Mirror, or Monaco

<k-code-editor> <!-- Init Code Here !--> </k-code-editor>
