Description

The <k-table> component is used to render data into a <table> using JavaScript.

Basic Usage

In HTML create a <k-table>, and then in JavaScript call the setData method passing in an array of objects containing your data.

<k-table id="contacts"></k-table>
<script type="module">
  document.getElementById('contacts').setData([
    {
      name: "Dustin",
      phoneNumber: "(111) 111-1111",
      emailAddress: "dustin@mailserver.com",
      gender: "Male"
    },
    {
      name: "Kayla",
      phoneNumber: "(222) 222-2222",
      emailAddress: "kayla@mailserver.com",
      gender: "Female"
    },
    {
      name: "Alexander",
      phoneNumber: "(333) 333-33333",
      emailAddress: "alex@mailserver.com",
      gender: "Male"
    },
    {
      name: "Amelia",
      phoneNumber: "(444) 444-44444",
      emailAddress: "amelia@mailserver.com",
      gender: "Female"
    },
  ]);
</script>

Custom Fields

Use the second paramter of setData to define custom fields.

<k-table id="movies"></k-table>
<script type="module">
  document.getElementById('movies').setData([
    {
      name: 'Good Will Hunting',
      year: 1997,
      lead: 'Matt Damon',
      budget: 10000000
    },
    {
      name: 'A Few Good Men',
      year: 1992,
      lead: 'Tom Cruise',
      prod: 'Castle Rock Entertainment'
    },
    {
      name: 'The Silence of the Lambs',
      year: 1991,
      lead: 'Jodie Foster',
      prequel: 'Red Dragon'
    }
  ],[
    {
      name: "name",
      label: "Name"
    },
    {
      name: 'year',
      label: 'Year'
    },
    {
      name: 'prod',
      label: 'Production Company'
    },
    {
      name: 'lead',
      label: 'Lead Actor / Actress'
    }
  ]);
</script>

Displaying Arrays

The table component can display arrays of strings or numbers by joining them with a comma and a space.

<k-table id="hobbies"></k-table>
<script type="module">
  document.getElementById('hobbies').setData([
    {
      name: "Dustin",
      hobbies: ["Reading", "Hiking", "Gaming"]
    },
    {
      name: "Kayla",
      hobbies: ["Cooking", "Traveling"]
    }
  ]);
</script>

Row Controls

The table component can display controls before or after each row. Each control is defined by an icon name and a callback function. All controls are combined into a single <td> with each control being a button.

<k-table id="contacts"></k-table>
<script type="module">
  document.getElementById('contacts').setData([
    {
      name: "Dustin",
      phoneNumber: "(111) 111-1111",
      emailAddress: "dustin@mailserver.com",
      gender: "Male"
    },
    {
      name: "Kayla",
      phoneNumber: "(222) 222-2222",
      emailAddress: "kayla@mailserver.com",
      gender: "Female"
    }
  ], false, {
    before: [
      {
        icon: 'edit',
        action: (record, table) => console.log('Edit', record, table)
      }
    ],
    after: [
      {
        icon: 'delete',
        action: (record, table) => console.log('Delete', record, table)
      }
    ]
  });
</script>

Constructor

Extends Component
new Table()
new Table(<Array>object records)
new Table(<Array>object records, <Array>object fields)
new Table(<Array>object records, <Array>object fields, object controls)

Parameters

<Array>object records

An array of objects that represent the records to be shown in the table.

<Array>object fields

An array of objects that represents the fields of the table, each object must contain a name and a label. If this parameter is omitted, the fields will be automatically generated from the first 100 records.

object controls

An object containing optional arrays of controls to be displayed before and/or after each row. Each control is defined by an icon name and a callback function. The controls object can contain the optional before and after properties, which are arrays of objects containing the icon (string) and action (function).

Example:

{
      before: [
        {
          icon: 'edit',
          action: (record, table) => console.log('Edit', record, table)
        }
      ],
      after: [
        {
          icon: 'delete',
          action: (record, table) => console.log('Delete', record, table)
        }
      ]
    }

Requirements

  • toTitleCase from string.js

Properties

fields: <Array>object

An array of objects containing a name and label that represent the fields to be displayed.

records: <Array>object

An array of objects containing the data that will be the records of the table, where the object keys match the field names.

controls: object

An object containing arrays of controls to be displayed before and/or after each row. Each control is defined by an icon name and a callback function. The controls object can only contain the before and after properties, which are arrays of objects containing the icon (string) and action (function).

Example:

{
      before: [
        {
          icon: 'edit',
          action: (record, table) => console.log('Edit', record, table)
        }
      ],
      after: [
        {
          icon: 'delete',
          action: (record, table) => console.log('Delete', record, table)
        }
      ]
    }

Methods

setData(<Array>object records, <Array>object fields): undefined

Sets the records and the fields and renders the table, if no fields are provided then they will be automatically generated from the keys of the first 100 records.

renderFields(): undefined

Renders the fields as table headers, this is called automatically by setData, you should never have to call it.

renderRecords(): undefined

Renders the records as table rows, this is called automatically by setData, you should never have to call it.

renderControls(<Array>object controls, object record): string

Renders the controls as table cells. Each control is defined by an icon name and a callback function. This is called automatically by renderRecords, you should never have to call it.
